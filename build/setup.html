Welcome! Let's get you set up.

1 Create a .env file

The app needs a configuration file to connect to your messaging platforms.

# Required: at least one bot token
TELEGRAM_TOKEN=your_telegram_bot_token

# AI Provider (pick one)
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...

# Optional
WEB_PASSWORD=your_dashboard_password
2 Place it in the right directory

Put the .env file next to the app, or use the CLI for guided setup:

npx alvin-bot setup
3 Restart the app

Quit and reopen Alvin Bot. The dashboard will load automatically once configured.
